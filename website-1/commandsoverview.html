Basic Commands used in linux
Command Usage Syntax
cd to change the working directory cd <directorypath>
ls to list files and directories in current folder ls [options] [filename]
pwd to see current working directory pwd [options]
mv to mv or rename files and directory mv <filename/folder> <newname>
rm to remove files rm <filename>
who to see current user logged in and using the system who [options] [filename]
cat to display files content on terminal cat [options] <filename>
mkdir to create directory mkdir [options] <directoryname>
rmdir to remove directory rmdir [options] <directoryname>
head prints the first 10 lines of the specified files head [options] [filename]
tail prints the last 10 lines of the specified files head [options] [filename]
grep filter searches a file for a particular pattern of characters grep [options] <pattern> [filename]
cp to copy file or directory to other location directory cp [options] <filename> <filename>
